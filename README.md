## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-org/ChatEDA-AcadBot.git
   cd ChatEDA-AcadBot
   ```
2. **Install dependencies**
   ```bash
   pnpm install
   ```
3. **Run the plugin**
   ```bash
   pnpm build
   ```
